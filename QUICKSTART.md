# üöÄ Quick Start Guide

## Step 1: Generate Icons (IMPORTANT!)

1. Open `generate-icons.html` in your browser
2. Click "Download All Icons"
3. Save the three PNG files to the `icons` folder:
   - icon16.png
   - icon48.png
   - icon128.png

## Step 2: Install the Extension

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" (toggle in top-right)
3. Click "Load unpacked"
4. Select the `AnyVideoConverter` folder
5. Done! The extension is now installed

## Step 3: Use the Extension

1. Click the extension icon in your Chrome toolbar
2. Upload a video or audio file
3. Choose output format and quality
4. Click "Convert"
5. Download your converted file!

## üìù Notes

- **First conversion may be slow** - FFmpeg needs to load (happens once)
- **All processing is local** - Your files never leave your computer
- **Large files take time** - Be patient, the browser does all the work
- **Memory intensive** - Close other tabs if needed

## ‚ùì Troubleshooting

**Extension won't load?**
- Enable Developer mode
- Check all files are present
- Reload extensions page

**Conversion fails?**
- Check file isn't corrupted
- Try different output format
- Ensure enough RAM available

**No icons showing?**
- Generate icons using `generate-icons.html`
- Make sure PNGs are in `icons` folder
- Reload the extension
